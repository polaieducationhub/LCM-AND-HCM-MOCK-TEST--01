# 🏆 LCM & HCF Special Mock Test

An attractive **Testbook Style Mock Test** made completely in **HTML, CSS & JavaScript**.

Designed for students preparing for:

✅ SSC CGL  
✅ SSC CHSL  
✅ Railway NTPC  
✅ Railway Group D  
✅ OSSC  
✅ OSSSC  
✅ Banking Exams  

---

## ✨ Features

- 📚 Topic: **LCM & HCF**
- ❓ Total Questions: **25**
- 🎯 Total Marks: **50**
- ✅ Correct Answer: **+2 Marks**
- ❌ Wrong Answer: **-0.5 Marks**
- ⏰ Total Time: **20 Minutes**
- 🌐 Multi Language Support:
  - English
  - हिन्दी (Hindi)
  - ଓଡ଼ିଆ (Odia)
